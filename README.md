R.M Imobiliária – Sistema de Orçamento (POO)
Descrição

Este projeto consiste em uma aplicação desenvolvida em Python para automatizar a geração de orçamentos de imóveis da imobiliária R.M. O sistema calcula o valor do aluguel com base no tipo de imóvel, aplica acréscimos e descontos e gera um arquivo CSV com as parcelas mensais.

Funcionalidades

Escolha entre apartamento, casa ou estúdio

Cálculo automático do aluguel

Aplicação de descontos e adicionais

Parcelamento do contrato em até 5 vezes

Geração de arquivo CSV com 12 parcelas

Tecnologias Utilizadas

Python 3

Biblioteca padrão csv

Estrutura do Projeto

rm_imobiliaria_poo.py → código principal do sistema

orcamento_rm.csv → arquivo gerado com as parcelas
